# Каталог книг

Веб‑приложение для управления каталогом книг и авторов с системой подписок на авторов.

## Возможности

- 📚 **Каталог книг** — просмотр с автоподгрузкой при скролле;
- 👤 **Каталог авторов** — список авторов с книгами, пагинация;
- 🔔 **Подписки на авторов** — получайте уведомления о новых книгах любимых авторов;
- 👤 **Регистрация и вход** — создание аккаунта пользователя;
- ⚙️ **Управление (для админов)** — полный CRUD для книг и авторов;
- 🖼️ **Обложки книг** — загрузка и управление изображениями;
- 🔔 **Уведомления** — персональные оповещения о новых книгах.

## Технологии

- **Backend:** Symfony 6.4, PHP 8.4;
- **Frontend:** Twig, Bootstrap 5, jQuery;
- **Database:** SQLite (для разработки);
- **ORM:** Doctrine.

## Установка

### Требования

- PHP 8.4;
- Composer;
- SQLite (или PostgreSQL для продакшена).

### Шаги установки

1. Клонировать репозиторий:

   ```bash
   git clone <repo-url>
   cd symfony
   ```

2. Установить зависимости:

   ```bash
   composer install
   ```

3. Настроить переменные окружения:

   ```bash
   cp .env .env.local
   # Отредактируйте .env.local при необходимости
   ```

4. Создать базу данных и запустить миграции:

   ```bash
   php bin/console doctrine:database:create
   php bin/console doctrine:migrations:migrate
   ```

5. Импортировать тестовые данные:

   ```bash
   php bin/console app:import-books
   ```

6. Создать тестового пользователя (опционально):

   ```bash
   php bin/console app:create-test-user
   ```

7. Запустить сервер:

   ```bash
   php -S localhost:8080 -t public
   ```

### Docker (опционально)

```bash
docker-compose up -d
```

## Использование

### Тестовые аккаунты

- **Админ:** `test@example.com` / `password123`;
- Зарегистрируйтесь самостоятельно для обычного аккаунта.

### Назначение админских прав

Для предоставления пользователю прав администратора выполните команду:

```bash
php bin/console app:add-admin
```

Эта команда назначает роль `ROLE_ADMIN` пользователю `test@example.com`.

### Основные маршруты

- `/` — Главная страница;
- `/book/` — Каталог книг;
- `/author/` — Каталог авторов;
- `/profile/` — Профиль пользователя;
- `/profile/notifications` — Уведомления;
- `/login` — Вход;
- `/register` — Регистрация.

### Роли

- **ROLE_USER** — обычный пользователь, может подписываться на авторов;
- **ROLE_ADMIN** — администратор, может управлять книгами и авторами.
